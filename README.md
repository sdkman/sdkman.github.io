### Installation

```
$ npm i
```

### Local Development

```bash
$ npm run dev
```

### Local Build

```bash
$ npm run build
```

### Production Build

```bash
$ NEXT_PUBLIC_BASE_URL=https://sdkman.io npm run build
```
